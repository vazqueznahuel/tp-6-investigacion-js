INTRODUCCION

¿Que es JavaScript?

Antes de ver y entender todos estas caracteristicas de JS (java script), deberiamos empezar por la de, ¿Que es Java Script?.

JS es un lenguaje de programacion interpretado, se suele utilizar como un lenguaje de sripting para paginas web, asi tambien como un lenguaje de programacion basada en prototipos, multiparadigmas, admitiendo estilos de programacion orientados a objetos, imperativos y funcionales.

Siendo de facil entendimiento y un potente lenguaje, con una aplicacion para controlar el comportamiento de paginas web como para estilizar y programar el comportamiento de las web segun los eventos que sucedan en la misma.

Netscape (lo que hoy conocemos como mozilla) en 1997 creo js, lo presento como estandar al European Computer Manufacturers Association, quedando con el nombre de ecma script como oficial, aun que todos hoy en dia lo conocen como js o java script, por esto a veces para referirse a este se utilizan las siglas ES por EcmaScript.

Punto numero 3

Variables

Las variables son declaradas de 3 maneras, ya que, cada de estas formas una tiene caracteristicas propias.

- Tener en cuenta que en JS NO HACE FALTA definir el tipo de variable ya que lo interpreta solo

Estas son:

Var:
Este tipo de variables funcionan como publicas, es decir, que el nombre y valor de las mismas puede ser utilizado en otros archivos.
Let:
En estas su valor y nombre se puede utilizar solo en el mismo archivo, esto significa que funciona como privada.
Const:
En estas variables el valor siempre es constante, una vez que se declara, no puede volver a ser modificadas.

Tipos de datos

En JS maneja 6 tipos de datos primitivos, estos son:

Boolean:
Tambien conocido como boleano, por su truccion al español, solamente maneja dos tipos de valores, True(verdadero) y False(falso).
Undifined:
Este se asigna a todas las variables a las que no se les atribuyo ningun valor a la hora de ser declaradas, en español se traduciria como indefinida.
Number:
Este representa a todos los numero como entero (Int), real (float), etc. Aunque tiene limitaciones y problemas con numeros demasiado grandes.
String:
Se utiliza para cadenas de caracteres, es decir, para textos, palabras, oraciones y demas.
Symbol:
Solo representa un solo caracter, es decir, que a la hora de usar este tipo de dato solo se le podra asignar un unico simbolo.
BigInt:
Este se utiliza para los grandes numeros, supliendo las desventajas que presentan los datos de tipo Number. Este es para numero enteros en el formato de presicion arbitrario.

Punto numero 6

Condicionales

If/else:
Esta condicional es simple, si se cumple la/las condiones de if entonces que se ejecuten las instrucciones de pasos que esta contiene y si no se cumplen, que suceda else y las instrucciones de paso que esta contiene.
Switch:
En esta hay 'x' cantidad de opciones que pueden suceder segun una variable, para que asi se ejecuten las instrucciones de pasos de la propia opcion que si se cumple, si no se cumple ninguna de las condiciones de las opciones entonces pasa esta otra cosa. Sirve para no tener que utilizar condicionales anidadas y asi ahorrar codigo.

Repetitivas

For:
Esta nos permite repetir instrucciones una cantidad de veces predeterminada por nosotros.
While:
En esta las instrucciones tambien se repiten, solo que al principio verifican que se cumplan ciertas condiciones para seguir ejecutando este codigo.
Do while:
Este funciona de manera parecida a While, solo que primero ejecuta las instrucciones para luego verificar las condiciones, si no se cumplen, las vuelve a ejecutar, de caso contrario simplemente sigue con el resto del algoritmo principal.

Eventos de JS

Onblur:
Si el elemento pierde el foco, por ejemplo, cuando hiciste click en un elemento de un formulario y luego pasaste a otro sin ingresar nada.
Onchange:
Este detecta si el elemento fue modificado.
Onclick:
Cuando se pulsa y se suelta el raton.
Ondclick:
Cuando se pulsa el raton dos veces.
Onfocus:
Cuando un elemento vuelve a tener el foco.
Onkeydown:
Cuando se pulsa una tecla y no se suelta.
Onkeypress:
Cuando se pulsa una tecla
Onkeyup:
Cuando se suelta una tecla pulsada.
Onload:
Cuando una pagina carga por completo.
Onmousedown:
Cuando se pulsa un boton del raton y no se suelta.
Onmousemove:
Cuando se mueve el raton
Onmouseout:
Cuando el raton no toca un elemento
Onmouseup:
Cuando el raton toca el elemento.
Onreset:
Cuando se inicializa el formulario.
Onresize:
Cuando se selecciona un texto.
Onselect:
Cuando se selecciono un elemento.
Onsumbit:
Cuando se evia un formulario.
Onunload:
Cuando se abandona la pagina.

Punto numero 10

A- Obtener datos del input

Hay 3 maneras para poder obtener los datos de un input de un formulario, estas son:

document.getElementById(id_name)
En este se utiliza el ID de la etiqueta para asi poder obtener los datos de el elemento con la misma.
document.getElementsByClassName('class_name')
Este funciona de manera muy simila al de ID, solo que con el atributo class.
document.querySelector('selector')
Con este simplemente deberemos de poner el nombre de la etiqueta, ya que este va a funcionar como selector, tal cual como en CSS, para asi obtener los datos del mismo.

B- Verificacion de datos del input

Para verificar los datos ingresados en los formularios, se utiliza mediante JS el manejador de eventos 'onsumbit' y desde el mismo se lleva a cabo el llamar a la funcion 'validar()' la cual se encarga de confirmar que se hayan colocados los datos correspondientes, a los campos que definimos.

C- Formulario de validacion que 'no sos un robot'

Un CAPTCHA es un pequeño algoritmo el cual ayuda a verficar que quien este ingresando datos o moviendose en el entorno web, no sea un robot o programa diseñado para tal accion. Para hacer uno primero se debe de crear un texto mediante JS, luego se muestra el texto al usuario de manera que solamente pueda ser leido si se vio el texto, para que asi el usuario ingrese el mismo en un elemento de entrada. Luego con una funcion se devolvera un valor que indique si es un robot o no mediante valores de true o false.
